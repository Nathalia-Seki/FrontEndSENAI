LISTAS NÃO ORDENADAS

  • Uma lista não ordenada, é onde a ordem dos itens não importam!
  • O termo UL vem de Unordered list!
  • type = Identificador dos itens de lista - formato deles, ou seja, bola, quadrado, etc
  • square = quadrado
  • circle = bola, mas não preenchida, somente circunferência
  • disc = padrão - que é a bolinha preenchida

LISTAS ORDENADAS

  • Uma lista ordenada, é onde a ordem e cada elemento é importante e deve ser seguida
  • O termo OL vem de Ordered List
  • O termo LI vem de List Item
  • type = Identificador dos itens de lista - formato deles, ou seja, numéro, letra, etc
  • a - alfabeto
  • i - romano
  • 1 - numeral
